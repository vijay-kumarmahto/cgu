## Key Design Principles

1. **Modularity**: Each detection type is self-contained
2. **Consistency**: All modules follow same API pattern
3. **Safety**: Atomic saves, drift detection, rollback capability
4. **Performance**: Hardware-aware optimization, cascade routing
5. **Usability**: Single entry point, automated setup, clear feedback

## Technology Stack

- **Framework**: PyTorch, Transformers (HuggingFace)
- **UI**: Gradio
- **Image**: ViT, SigLIP
- **Video**: GenConViT (ConvNeXt + Swin Transformer)
- **Audio**: Wav2Vec2
- **Audio Processing**: librosa, soundfile
- **Video Processing**: decord
